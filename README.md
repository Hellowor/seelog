### 项目介绍
* 与golang项目集成、提供浏览器实时查看日志的功能，类似 [tail -f xxx.log](https://www.cnblogs.com/fps2tao/p/7698224.html)
* 支持多浏览器同时访问
* 支持浏览器websocket断线重连
* 支持暂停、清屏、截图功能

### 集成方式
* 在项目中引入seelog, **go get github.com/xmge/seelog**
* 在代码中 执行 **seelog.See(logpath,port)**
* 在浏览器中访问 *http://host:port*
